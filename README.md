## Open Source Programs Beyond GSoC

While many students focus only on **Google Summer of Code (GSoC)**, there are several other impactful open-source programs that offer **excellent mentorship, global exposure, and even higher stipends**.

Below is a curated list you should definitely explore:

---

### 1. Google Summer of Code (GSoC)
🔗 https://summerofcode.withgoogle.com/

A global program by Google that brings new contributors into open source through paid summer projects.

---

### 2. Google Season of Docs (GSoD)
🔗 https://developers.google.com/season-of-docs

Focused on **technical writing**, documentation, and open-source sustainability.

---

### 3. Major League Hacking (MLH)
- **Fellowship**: https://fellowship.mlh.io/  
- **Prep Program**: https://fellowship.mlh.io/programs/prep  

Hands-on open-source experience with structured mentorship and peer learning.

---

### 4. Outreachy
🔗 https://www.outreachy.org/

Paid internships aimed at increasing diversity in open source and tech.

---

### 5. Hyperledger Foundation Mentorship
🔗 https://wiki.hyperledger.org/

Work on enterprise-grade blockchain and distributed ledger technologies.

---

### 6. LFX Mentorship (Linux Foundation)
🔗 https://mentorship.lfx.linuxfoundation.org/

Mentorship programs across major Linux Foundation projects.

---

### 7. Season of KDE
🔗 https://season.kde.org/

Contribute to KDE’s open-source desktop and application ecosystem.

---

### 8. FOSSASIA Internship Program
🔗 https://fossasiataipei.github.io/fossasia-internship/

Open-source internships under the FOSSASIA organization.

---

### 9. Summer of Haskell
🔗 https://summer.haskell.org/

A program dedicated to Haskell and functional programming contributions.

---

### 10. Cross Open Source Program (OSRE)
🔗 https://ucsc-ospo.github.io/osre/

University-led open-source research and engineering internships.

---

### 11. GitHub Octernships
🔗 https://education.github.com/students/octernships

Industry-backed open-source internships hosted on GitHub.

---

### 12. X.Org Foundation (EVoC)
🔗 https://www.x.org/wiki/XorgEVoC/

Contribute to core graphics and windowing systems used in Linux.

---

### 13. Microsoft Reinforcement Learning Open-Source Fest
🔗 https://www.microsoft.com/en-us/research/academic-program/rl-open-source-fest/

Work on reinforcement learning projects under Microsoft mentorship.

---

✨ **Pro Tip:**  
GSoC is great — but these programs can be equally (or more) rewarding in terms of learning, impact, and long-term career growth.
